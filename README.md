# SDL2
Simple DirectMedia Layer 2.0 for Amiga 68k RTG

WIP port of SDL2 to Amiga 68k that I've been working on for the past couple of weeks that used the OS4 port as a starting point (thanks guys), most things are working I think.

I'll slowly add more functionality when I get the time.   I did this library to enable me to update some of my old SDL 1.2 ports like Exult and FreeSynd which are now SDL2 only.

For performance and simplicity, only 24/32-bit RTG modes are supported.

This library will only be of practical use running on a PiStorm 4.
