# SDL2
Simple DirectMedia Layer 2.0 for Amiga 68k RTG

WIP port of SDL2 to Amiga 68k that I've been working on for the past couple of weeks, most things are working I think.

I'll slowly add more functionality when I get the time.   I did this library to enable me to update some of my old SDL 1.2 ports like Exult that are now SDL2 only.

For performance and simplicity, only 24/32-bit RTG modes are supported.
